Introduction

The Drupal-Brightcove connector for Drupal 11 updates the previous version to all you to take advantage of the new features in Drupal 11. While Drupal 11 does not offer significant new features over Drupal 9, it does introduce performance enhancements and update underlying software subsystems to increase security, stability, and maintainability

Installation and documentation

To install the connector for Drupal 11, please follow the instructions on the Drupal site.
